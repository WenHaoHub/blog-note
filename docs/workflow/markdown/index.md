Markdown 是一种轻量级标记语言，广泛应用于编写文档、博客、README 文件等场景。本文将为你提供 Markdown 语法的详细教学，帮助你快速上手并享受书写优雅文档的乐趣。

## 1. 标题

在 Markdown 中，你可以使用`#`符号来表示标题。一个`#`表示一级标题，两个`#`表示二级标题，以此类推，最多支持六级标题。

示例:

```
# 一级标题
## 二级标题
### 三级标题
```

## 2. 文本样式

Markdown 提供了多种文本样式，让你的文档更加丰富和易读。

- **加粗文本:** 使用两个`**`或`__`将文本包围起来，实现加粗效果。
- **斜体文本:** 使用一个`*`或`_`将文本包围起来，实现斜体效果。
- **删除线文本:** 使用两个`~~`将文本包围起来，实现删除线效果。

示例:

```markdown
**加粗文本**
_斜体文本_
~~删除线文本~~
```

## 3. 列表

在 Markdown 中，你可以创建有序列表和无序列表。

- **有序列表:** 使用数字后跟一个英文句点`.`+空格，然后添加列表项内容。 md 编辑器快捷键`ctrl+shift+[`
- **无序列表:** 使用`-`、`+`或`*`加上一个空格，然后添加列表项内容。 md 编辑器快捷键`ctrl+shift+]`

示例:

```markdown
1. 有序列表项 1
2. 有序列表项 2
   - 无序列表项 A
   - 无序列表项 B
```

## 4. 链接与图片

在 Markdown 中，你可以插入链接和图片到文档中。

- **链接:** 使用`[]`表示链接显示的文本，紧接着使用`()`表示链接的地址。
- **图片:**

  1. 使用`img`标签 更好的控制大小

  2. 使用`![]`表示插入图片，括号中第一个引号内为图片的描述，第二个引号内为图片的地址。

示例:

<img title="示例图片" src="http://39.98.94.11:8080/image/logo.png" width="45%">

![示例图片](http://39.98.94.11:8080/image/logo.png)

```markdown
[点击这里访问 GitHub](https://github.com/)
![示例图片](http://39.98.94.11:8080/image/logo.png)
<img title="示例图片" src="http://39.98.94.11:8080/image/logo.png" width="45%">
```

## 5. 引用与代码块

Markdown 允许你引用他人的文字或展示代码块。

- **引用:** 使用`>`符号表示引用，可以嵌套多个引用。
- **代码块:** 使用三个反引号```````包围代码块，也可以指定代码语言以获得语法高亮效果。

示例:

> 这是一段引用的文字。

## 6. 表格

Markdown 支持创建表格，让你的数据呈现更加整齐清晰。

- 使用`|`符号分隔不同的列，使用`-`符号分隔表头和表格内容。
- 第一行为表头，第二行为分隔表头和表格内容的行，后续行为表格内容。

示例:

| 姓名 | 年龄 |
| ---- | ---- |
| 张三 | 25   |
| 李四 | 30   |
